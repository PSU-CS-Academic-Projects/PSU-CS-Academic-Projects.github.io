# PSU-CS-Academic-Projects Portfolio

A premium, open-source GitHub Pages portfolio for the PSU CS Academic Projects organization.

## Tech Stack
- Vanilla HTML / CSS / JavaScript — no build step
- GitHub API for live data
- Hosted on GitHub Pages

## Features
- Live GitHub API integration (repos, members, org stats)
- Animated hero with counter stats
- Filterable project card grid with search
- Project detail modal
- Members section
- Responsive: mobile → tablet → desktop
- Graceful fallback if API rate-limited

## Local Development
Just open \index.html\ in your browser — no server needed.
